The most minimal ArcFace face comparison tutorial project. / 本项目是最精简的 ArcFace 人脸比对教学项目。

This project removes all unnecessary complexity, retaining only the core implementation to help beginners quickly understand and implement ArcFace. / 本项目去除了所有不必要的复杂依赖和代码，只保留核心实现，让初学者能在最短时间内理解 ArcFace 的原理并上手实践。

## Features ## 特性

- Extremely simple code structure, clear at a glance / 极简代码结构，一目了然  
- Pure TensorFlow 2.x implementation, no need for large frameworks like InsightFace / 纯 TensorFlow 2.x 实现，无需 InsightFace 等大型框架  
- Supports multiple platforms: Windows and Linux / 支持 Windows 和 Linux 多平台  
- Provides complete training + inference pipeline / 提供完整的训练 + 推理流程  
- Includes optional pre-trained model weights / 包含预训练模型权重（可选下载）  
- Detailed Chinese comments for easy learning and teaching / 详细中文注释，便于教学和学习  

## Requirements ## 环境要求

- Python 3.6 or higher / Python 3.6 或更高版本  
- TensorFlow 2.6.0 or higher / TensorFlow 2.6.0 或更高版本  
- Supports Windows / Linux (macOS not tested but theoretically compatible) / 支持 Windows / Linux（macOS 未测试，但理论支持）
